# ClearingHouse ClearingHouse is a protocol that allows for asset issuance and p2p trading on the [Viacoin](http://viacoin.org) blockchain. It was released on 11th August 2014 although this website is under construction. You can read more about it on the Viacoin Blog: http://blog.viacoin.org/2014/08/11/1-clearinghouse.html ## Clearwallet [Clearwallet.io](https://clearwallet.io) is a web implementation that takes advantage of the ClearingHouse protocol.
