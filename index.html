# ClearingHouse ClearingHouse is a protocol that allows for asset issuance and p2p trading on the [Viacoin](http://viacoin.org) blockchain. It was released on 11th August 2014. (website is under construction). ## Clearwallet [Clearwallet.io](https://clearwallet.io) is a web implementation that takes advantage of the ClearingHouse protocol.
